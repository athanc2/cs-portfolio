HOW TO PAGE

For my about me page I started with the <h1> tag for the first heading with a red ridge border and a pink background.

In my Introduction, I used the <p> tag with <b> tag to make it bold and I added a video game gif just to make it look cooler.

For my second heading (What I have learned), I used <h2> tag with double green border.

For my List, I started with <ul> with an id of table so I can edit it easier, for the list content I used <li> so it is bullet pointed and I used a gray ridge border with an orange background for it.
